== 說明文件

這是一個線上課程心得分享平台，能夠很簡單地讓同學們值估店分享對於課程的觀感！

== 功能

* 課程（新增、刪除、修改）
* FB 登入（製作中）
* 檢舉課程 (製作中)

== 使用的 Function

* Bootstrap 4 aplha
* Devise FB Login

== 使用此專案方法

請先使用以下方法下載至本地端：
<tt>$ git clone https://github.com/acer1456/classComment.git</tt>

進入專案資料夾：
<tt>$ cd classComment</tt>

進行 migrate 建立資料庫：
<tt>$ rake db:migrate</tt>

打開伺服器即可開始測試使用：
<tt>$ rails s</tt>
